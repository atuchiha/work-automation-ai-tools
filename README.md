🚀 目的

このリポジトリの目的は、
「AI × プログラミングで、誰でも使える業務効率化ツールを作ること」
です。

特に、

毎日の単純作業を減らしたい

仕事のスピードを上げたい

データ整理やメール作業などを自動にしたい

人の手作業によるミスをなくしたい

といった課題を解決するツールの開発を目指しています。

● 生成AI

● VBA

● Python

など幅広い自動化に利用
